Just click the Testdrive link in the right column and see WebYaST loading in your browser.

WebYaST allows you to manage your Linux workstation or server easily using a web interface. It strives for simplicity and focuses on tasks admins commonly perform (like user management or software updates). WebYaST is especially useful when you can't access your machine via SSH or don't want to remember all the commands.

What can WebYaST do?

  • Check system health and status
  • Add, remove and edit users
  • Update software
  • Manage system services
  • Configure network settings
  • Configure firewall
  • And much more...
