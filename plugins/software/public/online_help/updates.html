Managing System Updates

The Updates WebYaST module displays a list of available system updates (sometimes called patch). Use the list to select the updates you want to install.

To update the system, follow these steps:

  1. If the system is up to date, click Back.

  2. Use the checkboxes to select the updates that you want to install.

  3. Click Update.

Alternatively, you can update your appliance from the command line with ssh and the zypper up command.
