Configuring Mail Settings

The Mail Settings WebYaST module allows you to configure a server for outgoing mail (SMTP). To configure the SMTP server, follow these steps:

  1. In Outgoing mail server, enter the URL of the SMTP server to be used.

    You may also specify an alternate port number (the default is 25), e.g. relay.example.com:26. If the host is not mail exchanger, then you have to use square brackets around the host name to prevent DNS MX lookups, e.g. [relay.example.com]:26.

  2. In Use secure connection (TLS), choose whether encryption (TLS) should be used. The available options are: Disabled, If available, and Always.

  3. Enter your credentials for the SMTP server: User name and Password. Confirm the password by entering it in Confirm password.

  4. You can test the mail settings by entering an e-mail address in Send test mail to and pressing Send Mail. The test mail will be also sent when clicking Save.

  5. Click Save
