Kerberos Authentication

The Kerberos WebYaST module allows you to enable authentication via Kerberos client.

You can find more information about Kerberos in SUSE Security Guide.

To enable Kerberos authentication, follow these steps:

  1. Check the Enable Kerberos Authentication option.

  2. Set the Default Domain, Default Realm, and KDC Server Address as provided by your administrator.

  3. Click Save.
