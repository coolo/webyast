Managing System Services

The System Services WebYaST module allows you to manage system services available on the target system. To manage the system services, follow these steps:

  1. A list of available system services is shown. Using the Filter Services drop-down menu, you can limit the list to show only the running, not running, dead, started on boot, or not started on boot services. Using the Quick Search, you can search services according to their names and descriptions.

  2. For each service, a status icon, name and a short description are shown. You can expand the information shown by clicking the little triangle on the left. A longer description, detailed status information and Start/Stop, Restart, and Start on Boot/Do Not Start on Boot buttons are shown in the expanded view.

    To change both the current status and the on-boot status (enable or disable for the current runlevel) press Start or Stop button.

    To restart a service without changing its on-boot status, press Restart.

    To change only the on-boot status (enable or disable for the current runlevel) press Start on Boot orÂ Do Not Start on Boot button.

    The result of the requested status change is shown under the status buttons.

  3. If you are finished, click Back.

Configuring the List of System Services

The list of services available in the System Services module is configurable. For more information, see the vendor documentation.
